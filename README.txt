本工程为cling库作者写的Android demo，用于学习和演示Android下upnp java库cling。

我在原来的工程中添加了libs中的Jar库文件，修改了src下的两个路径（否则导致路径正确，即使能够编译出apk文件，也不能够正常运行）。

直接导入该工程到Eclipse中则可。
